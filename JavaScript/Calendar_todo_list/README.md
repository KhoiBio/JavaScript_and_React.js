# Calendar To-Do App (Vanilla JavaScript)

A simple **calendar-based to-do app** built with **plain JavaScript, HTML, and CSS**, allowing you to:

- Navigate months  
- Click a day to view/add/delete tasks  
- Mark tasks as completed with a checkbox  
- Persist tasks using **localStorage**  

This is lightweight, fully client-side, and **portfolio-ready**.

---

## Features

- **Calendar with month navigation**  
- **Add / Delete / Complete tasks**  
- Completed tasks appear **strikethrough**  
- Tasks persist across page reloads using **localStorage**  
- Days with pending tasks are bold on the calendar  

---

## How to Run (No Install Needed)

1. **Download the project files**  
   - Ensure you have the following:  
     - `vanila_javascript_todo_list.html` (contains HTML + JavaScript)  
    
2. **Open `vanila_javascript_todo_list.html` in a web browser**  
   - Double-click the file or open with **Chrome / Edge / Firefox**  
   - The app will load instantly with the current month displayed  

3. **Use the app**  
   - Click a day to **view/add/delete tasks**  
   - Check the checkbox to **mark a task as completed**  
   - Use **Prev / Next buttons** to navigate months  
   - Completed tasks will have **strikethrough** and stay saved after refreshing the page  

---

## LocalStorage

- All tasks are saved in your browser using `localStorage`  
- Data persists even if you **close and reopen the browser**  
- Each task includes a **completed status** and text  

---

## Notes
